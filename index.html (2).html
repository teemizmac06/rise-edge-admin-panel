How To Use
Complete guide to managing your RiseEdge website.
📌 Adding a New Paid Spotlight Author
1Author pays the reservation fee → you get payment confirmation
2Go to Spotlight → click Add New Author
3Upload their real photo (or paste a URL), fill in name, bio, genre, event date, Amazon link
4Toggle Set as Featured if this is the next upcoming event
5Click Save — profile goes live on website instantly
⚠️ Why changes must be on the same domain
Both riseedge-bookclub.html and riseedge-admin.html MUST be deployed together to the same Vercel project. This is because localStorage is domain-specific — so vercel.app/admin and vercel.app share the same storage, but localhost and vercel.app do not. If you edit on localhost and view on Vercel, changes won't appear.
🔑 Changing your password

Open riseedge-admin.html in a code editor. Search for riseedge and admin2024 and replace with your preferred username and password. Then re-deploy to Vercel.

📸 Best practices for photos

For author photos, use direct URLs from: Imgur (free, paste image URL), Cloudinary (free tier), or Google Drive (set file to public, use direct link). Uploaded files are converted to base64 and may be large — URLs are preferred for performance.
